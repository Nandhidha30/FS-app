Put all product images in: `frontend/public/images/products/`

## How to Download and Save

1. Click on any Google Images search link above
2. Right-click on an image you like
3. Select "Save image as..."
4. Save with the exact filename (e.g., "golden-barrel-cactus.jpg")
5. Place in `frontend/public/images/products/` folder

## Alternative: Use Placeholder Images

If you don't want to download images, the app will show plant emojis as placeholders.
